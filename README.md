# Career Recommender System

This project recommends suitable career paths based on user skills and interests using Machine Learning.

## Features
- Skill based career recommendation
- User input interface
- ML model prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Flask

## How to Run

1. Clone the repository
git clone https://github.com/yourusername/career-recommender-system.git

2. Install requirements
pip install -r requirements.txt

3. Run the project
python app.py
